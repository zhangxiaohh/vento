## 👋 欢迎来到我的个人博客

这是我的个人技术博客，记录我的学习历程、技术分享和生活感悟。

## 🎯 博客目标

- 分享前端开发技术和最佳实践
- 记录个人学习成长历程
- 交流编程心得和解决方案
- 建立个人知识库和技术档案

## � 博客内容

- **前端技术**：React、Vue、TypeScript、JavaScript 等
- **开发工具**：Git、VS Code、构建工具等
- **编程思想**：设计模式、代码优化、性能提升等
- **生活感悟**：学习方法、时间管理、职业发展等

## 🔧 技术栈

- **框架**：Astro
- **语言**：TypeScript、JavaScript、Markdown
- **样式**：Tailwind CSS
- **国际化**：支持多语言切换
- **部署**：GitHub Pages

## � 如何浏览

1. **首页**：查看最新文章和精选内容
2. **文章列表**：浏览所有文章
3. **标签**：按标签筛选相关文章
4. **归档**：按时间查看历史文章
5. **关于**：了解博主信息
